# Sales agent adoption report

The Sales agent adoption report is an essential tool for sales leaders looking to understand the usage of the Sales agent within their organization. Using this Power BI template, you gain valuable insights into how different groups and apps are utilizing the Sales agent. You can also identify key actions contributing to sales tasks, and explore research findings on its adoption.
 
This report is divided into several sections that provide a comprehensive view of usage metrics, trends over time, and practical guides on getting started with the Sales agent.
 
To learn more about setting up and using this report, go to [Sales agent adoption report](/viva/insights/advanced/analyst/templates/copilot-for-sales-adoption).
